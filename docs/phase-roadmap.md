# Phase Roadmap

## Phase 1: Rapid MVP Prototype

Current phase. Build a local web app with a simple chat assistant, OpenAI API integration, a local FYEC100 knowledge base, guardrails, academic integrity guidance, and basic documentation.

## Phase 2: Enterprise Integrated Deployment

Future phase. Prepare for institutional deployment with authentication, hosting, monitoring, privacy review, Moodle integration, Banner-aware support pathways, production governance, analytics, and support procedures.

## Phase 3: Advanced AI Learning Ecosystem

Future phase. Expand toward personalized learning support, proactive student success nudges, richer analytics, multilingual support, adaptive learning resources, and broader student services integration.
