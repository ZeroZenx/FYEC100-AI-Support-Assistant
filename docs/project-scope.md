# Project Scope

## Project

FYEC100 AI Support Assistant

## Phase 1 Objective

Build a local MVP prototype that demonstrates an AI-powered academic support assistant for FYEC100, a first-year elective course required by first-year students.

## In Scope for Phase 1

- Local Next.js web app
- Simple student-facing chat interface
- OpenAI API route for responses
- Local Markdown knowledge base
- FYEC100 sample course content
- Academic integrity guardrails
- Refusal behavior for full assignment writing
- Refusal behavior for grading claims
- Fallback responses when information is not in the knowledge base
- Home, Chat, About, and Roadmap pages
- GitHub-ready documentation

## Out of Scope for Phase 1

- Database
- Authentication
- Moodle integration
- Banner integration
- Production hosting
- Student record access
- Official grading
- Automated assignment submission

## Success Criteria

The prototype should run locally on a Mac at `http://localhost:3000`, allow users to ask FYEC100-related questions, and demonstrate a responsible path toward enterprise deployment.
